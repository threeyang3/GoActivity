## 视图十二：演出放映（Grid）

| 项 | 值 |
|---|---|
| 类型 | 表格（Grid） |
| 名称 | 演出放映 |
| 过滤 | `活动类型` = `演出·放映` |
| 排序 | 开始时间 ↑ |

### 显示字段

- 标题（280px）
- 演出类型（100px）

- 标签（120px）
- 摘要（280px）

## 操作步骤

1. 打开"活动库" Bitable
2. 右上 `+ 视图` → 选类型 → 配置名称、过滤、排序、分组
3. 按上面 7 个视图逐一创建
4. 收藏最常用的 3 个：演出放映 / 讲座论坛 / 比赛征稿
